## 🏗 Featured Project

### 💳 Credit Card Fraud Detection — ML Classification Pipeline

> End-to-end fraud detection system trained on a real-world, highly imbalanced transaction dataset. Handles the core challenge of financial ML: detecting rare fraud events without drowning legitimate users in false positives.

**What it does:**
- Cleans and preprocesses raw transaction data for reliable model input
- Applies **SMOTE** to synthetically balance the minority fraud class before training
- Trains a **Random Forest Classifier** optimised for precision-recall on skewed data
- Achieves strong fraud detection with meaningfully reduced false positive rate
- Produces full visual analysis: confusion matrix, feature importance, class distribution

**The hard problem solved:** In fraud datasets, fraudulent transactions are often <0.2% of all records. A naive model predicts "not fraud" every time and hits 99.8% accuracy — but catches zero fraud. This pipeline addresses that directly through SMOTE resampling and threshold-aware evaluation.

**Stack:** `Python` · `Scikit-learn` · `Pandas` · `NumPy` · `Matplotlib` · `Seaborn` · `Google Colab`
